# PythonShootGame

A simple shoot game written in Python.
파이썬으로 작동하는 간단한 게임입니다.

# Introduction

This project only include two simple .py files: 
이 프로잭트는 단 두 개의 파이썬 파일로 작동합니다

1. mainGame.py: The initialization and main loop of the game.
2. gameRole.py: Class of the game role.

1. mainGame.py : 게임의 초기화 및 메인 루프.
2. gameRole.py : 게임 룰의 클래스.

# Requirement

1. Python 2.7
2. Python-Pygame
  
# How To Start Game
  
```bash
$ python mainGame.py
```
